# List<对象>请求 s

[]为一个 List 集合，{}为一个对象 

```json
[
  {
    "字段1": 1,
    "字段2": "ssss",
    "字段3": 1,
    "字段4": 1,
    "字段5": "ss"
  },
  {
    "字段1": 1,
    "字段2": "ssss",
    "字段3": 2,
    "字段4": 3,
    "字段5": "sdsss"
  }
]
```

# List<Long>请求

```json
["1", "2", "3"]
```
